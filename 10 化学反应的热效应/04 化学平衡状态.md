# 化学平衡状态
## 一、可逆反应
1. 定义：同一条件下，既能向正反应方向进行，同时又能向逆反应方向进行的化学反应。
2. 特点：
    - 二同：相同条件下、正逆反应同时进行。
    - 一小：反应物与生成物同时存在，任一组分转化率小于100%。
3. 表示：化学方程式中用“⇌”表示。
4. 核心特征：双向性、双同性、共存性，反应物无法100%转化。

## 二、化学平衡状态
1. 定义：一定条件下，可逆反应体系中，正反应速率与逆反应速率相等，反应物和生成物的浓度均保持不变，体系组成不随时间改变，此时的状态为化学平衡状态（简称化学平衡）。
2. 建立过程：
    - 从正反应开始：反应物浓度降低，正反应速率减小；生成物浓度升高，逆反应速率增大，最终v(正)=v(逆)。
    - 从逆反应开始：生成物浓度降低，逆反应速率减小；反应物浓度升高，正反应速率增大，最终v(正)=v(逆)。
    - 平衡建立后，体系看似“静止”，实则正逆反应持续进行，属于动态平衡。
3. 特征：
    - 研究对象：可逆反应。
    - 动态平衡：v(正)=v(逆)≠0。
    - 浓度不变：反应物和生成物的浓度、百分含量保持不变。
    - 条件依存：条件改变，平衡可能移动，新条件下建立新平衡。

## 三、化学平衡状态判断（以\(m A(g)+n B(g) \rightleftharpoons p C(g)+q D(g)\)为例）
### （一）判断核心
1. 正逆反应速率之比等于化学计量数之比（需体现“一正一逆”）。
2. 随反应进行的“变量”不再变化（变量指随反应正向/逆向进行而改变的物理量）。

### （二）具体判断依据
| 类型 | 判断依据 | 是否为平衡状态 |
| --- | --- | --- |
| 混合体系中各成分的含量 | ①各物质的物质的量或物质的量分数一定 | 是 |
| | ②各物质的质量或质量分数一定 | 是 |
| | ③各气体的体积或体积分数一定 | 是 |
| 正逆反应速率的关系 | ①单位时间内消耗m mol A，同时生成m mol A（v正=v逆） | 是 |
| | ②单位时间内消耗n mol B，同时生成p mol C（v正与v逆无必然关系） | 否 |
| | ③v(A)∶v(B)∶v(C)∶v(D)=m∶n∶p∶q（仅体现正反应速率比例，与平衡无关） | 否 |
| | ④单位时间内生成n mol B，同时消耗q mol D（均表示v逆） | 否 |
| 压强 | ①m+n≠p+q时，总压强一定（其他条件不变） | 是 |
| | ②m+n=p+q时，总压强一定（其他条件不变） | 否 |
| 平均相对分子质量（\(\overline{M_r}\)） | ①m+n≠p+q时，\(\overline{M_r}\)一定 | 是 |
| | ②m+n=p+q时，\(\overline{M_r}\)一定 | 否 |
| 气体密度 | ①m+n≠p+q，恒温恒压时，密度保持不变 | 是 |
| | ②m+n=p+q，恒温恒压时，密度保持不变 | 否 |
| 颜色 | 反应体系内有色物质的颜色一定 | 是 |
| 绝热条件下体系的温度 | 体系温度不变 | 是 |
| 绝热条件下体系的压强 | m+n=p+q时，压强不变 | 是 |

## 四、化学平衡常数
### （一）浓度商
1. 定义：任意时刻，生成物浓度幂之积与反应物浓度幂之积的比值，用Q表示。
2. 表达式：对于可逆反应\(m A(g)+n B(g) \rightleftharpoons p C(g)+q D(g)\)，\(Q=\frac{c^p(C) \cdot c^q(D)}{c^m(A) \cdot c^n(B)}\)。

### （二）化学平衡常数（K）
1. 定义：温度一定时，可逆反应达到平衡状态时，生成物浓度幂之积与反应物浓度幂之积的比值（常数）。
2. 表达式：同上述反应，\(K=\frac{c^p(C) \cdot c^q(D)}{c^m(A) \cdot c^n(B)}\)（固体、纯液体不列入表达式）。
3. 意义：K值越大，反应进行越完全，反应物转化率越高；\(K>10^5\)时反应基本完全，\(K<10^{-5}\)时反应难发生。
4. 与方程式的关系：
    - 正逆反应的K互为倒数（\(K_{正}=\frac{1}{K_{逆}}\)）。
    - 方程式乘以系数x，K变为原来的x次方。
    - 两方程式相加，总反应K等于两分反应K的乘积（\(K_{总}=K_1 \cdot K_2\)）。
5. 影响因素：仅受温度影响，与浓度、压强、催化剂无关。
6. 应用：判断反应方向
    - Q<K：反应正向进行（v正>v逆）。
    - Q=K：反应达平衡（v正=v逆）。
    - Q>K：反应逆向进行（v正<v逆）。

## 五、化学平衡移动
### （一）概念与实质
1. 概念：平衡后改变条件，原平衡被破坏，一段时间后建立新平衡的过程。
2. 实质：条件改变导致v正≠v逆，各组分百分含量改变。

### （二）勒夏特列原理
改变影响平衡的条件之一（浓度、压强、温度），平衡将向着减弱该改变的方向移动（“减弱”而非“抵消”）。

### （三）外界条件对平衡的影响
| 改变条件（其他不变） | 平衡移动方向 | 说明 |
| --- | --- | --- |
| 浓度 | 增大反应物浓度/减小生成物浓度 | 正向移动 |
| | 减小反应物浓度/增大生成物浓度 | 逆向移动 |
| 压强（有气体参与） | m+n≠p+q，增大压强 | 向气体分子数减小方向移动 |
| | m+n≠p+q，减小压强 | 向气体分子数增大方向移动 |
| | m+n=p+q，改变压强 | 平衡不移动 |
| 温度 | 升高温度 | 向吸热反应方向移动 |
| | 降低温度 | 向放热反应方向移动 |
| 催化剂 | 加入催化剂 | 平衡不移动（同等程度改变v正、v逆） |

### （四）惰性气体的影响
1. 恒温恒容：充入惰性气体，总压强增大，但各组分浓度不变，平衡不移动。
2. 恒温恒压：充入惰性气体，容器体积增大，各组分浓度同倍数减小，等效于减压，平衡按减压规律移动。

## 六、等效平衡
### （一）定义
同一可逆反应，在一定条件（恒温恒容/恒温恒压）下，不同投料方式达到平衡时，相同组分的百分含量（体积、物质的量、质量分数）相等的平衡状态。

### （二）判断方法：极值转换法（“一边倒”）
将反应物或生成物按计量数完全转化为另一侧，比较投料量关系。

### （三）分类与结论
1. 恒温恒容：
    - 一边倒后投料量完全相等（等量等效）：平衡时各物质浓度、百分含量均相等。
    - 一边倒后投料量成比例（气体分子数不等反应）：等效为加压/减压，平衡移动，百分含量不同。
    - 一边倒后投料量成比例（气体分子数相等反应）：平衡不移动，百分含量相等。
2. 恒温恒压：
    - 一边倒后投料量完全相等或成比例：均为等效平衡，百分含量相等。

### （四）转化率与热量关系
- 若两过程为等效平衡且反应方向相反，转化率之和为1（\(\alpha_1+\alpha_2=1\)）。
- 热量关系：\(|Q_1|+|Q_2|=|\Delta H|\)（Q为实际放热/吸热，\(\Delta H\)为反应焓变）。
