# 从喷泉实验到气体溶于水浓度的计算问题

## 导入
喷泉实验的本质是气体溶于水形成的压强差，那我们是否考虑过：一定体积的气体溶于水，形成溶液的浓度？

以下就以喷泉实验为情境，讨论从基础到复杂的几种情况，条件均设为标准状况

## 简单的几种情况：
### 仅 $\ce{NH3}$

> 描述：容积为 $\ce{aL}$ 的烧瓶中盛满氨气，现进行喷泉实验，求实验后得到的溶液浓度

解答：$\ce{n(NH3) = \frac{a}{22.4}mol \implies c= \frac{n(NH3)}{V} = \frac{\frac{a}{22.4}}{a} = \frac{1}{22.4}M}$

总结：上面讨论了完全溶于水的气体形成溶液的浓度，对于 $\ce{NH3, HCl}$ 等均可使用

### 仅 $\ce{NO2}$

描述同上一题

这个问题要多考虑的一点是：$\ce{NO2}$ 溶于水后会放出 $\ce{1/3}$ 体积的 $\ce{NO}$ (方程式 $\ce{3NO2 + H2O = 2HNO3 + NO}$)

因此实际溶于水的气体体积只有 $\ce{2/3aL}$，由于烧瓶内会产生 $\ce{1/3}$ 的空隙，溶液的体积也是 $\ce{2/3aL}$，据此解答

解答：$\ce{n(HNO3) = 2/3 * \frac{a}{22.4}mol \implies c= \frac{n(HNO3)}{V} = \frac{2/3 * \frac{a}{22.4}}{2/3a} = \frac{1}{22.4}M}$

总结：这里讨论了溶于水会放出其他气体的情况，还是非常基本的问题，只需牢牢抓住  $\ce{c = \frac{n(溶质)}{V(溶液)}}$ 即可作答

### 深入思考

>  若将上面的条件换为“容积为 $\ce{aL}$ 的烧瓶中盛有 $k\ce{aL}$ 氨气($k∈(0,1)$)”，得到的溶液浓度会改变吗？

**不会**，原理很简单，只充了 $k\ce{aL}$ 氨气，喷泉是不会到顶的，由此：$\ce{c= \frac{n(NH3)}{V} = \frac{\frac{ka}{22.4}}{ka} = \frac{1}{22.4}M}$

下面我们考虑稍微复杂的情况

## 多种气体以比例溶于水：

### $\ce{NO}$ 与 $\ce{O2}$ 混合

#### V(NO)/V(O<sub>2</sub>) = 4:3

> 写出方程式：$\ce{4NO + 3O2 + 2H2O -> 4HNO3}$

解析：该体积比(4:3)下 $\ce{NO}$ 与$\ce{O2}$ 恰好完全反应，因此：$\ce{V_{liquid} = aL}$ 

现在我们要求硝酸物质的量，观察到上述方程式中硝酸计量数等于一氧化氮，而一氧化氮占总气体的体积：$\ce{V_{NO} = 4/7aL}$

因此有：$\ce{n(HNO3) = n(NO) = \frac{4/7a}{22.4}mol}$

从而求出浓度：$\ce{c = \frac{n(HNO3)}{V_{liquid}} = \frac{4/7a* \frac{1}{22.4}}{a} = 4/7 * \frac{1}{22.4}M}$

***

#### V(NO)/V(O<sub>2</sub>) = 4:4

$\ce{4NO + 3O2 + 2H2O -> 4HNO3}$

类似于上面的思路，我们有 $\ce{n(HNO3) = n(NO) = \frac{1/2a}{22.4}mol}$

八份气体中有一份多余的氧气，所以 $\ce{V_{liquid} = 7/8aL}$

所以 $\ce{c = \frac{n(HNO3)}{V_{liquid}} = \frac{1/2a* \frac{1}{22.4}}{\frac{7}{8}a} = 4/7 * \frac{1}{22.4}M}$

***

#### V(NO)/V(O<sub>2</sub>) = 5:3

> 举这个例子是为了说明做题一定不能死板套公式，如本题中 $\ce{HNO3}$ 物质的量就不能看作 $\ce{5/8a*\frac{1}{22.4}mol}$ ，因为5份一氧化氮中只有4份能够反应生成硝酸

$\ce{n(HNO3) = n(NO_{reacted}) = \frac{4/8a}{22.4}mol \\V_{liquid} = 7/8aL}$

$\ce{\implies c = 4/7 * \frac{1}{22.4}M}$

***

### $\ce{NO2}$ 与 $\ce{O2}$ 混合

#### V(NO<sub>2</sub>)/V(O<sub>2</sub>) = 4:1

> 写出方程式：$\ce{4NO2 + O2 + 2H2O -> 4HNO3}$

解析：题给条件下两气体恰好完全反应，从而：$\ce{V_{liquid} = aL}$

显然 $\ce{n(HNO3) = n(NO2) = \frac{4/5a}{22.4}mol}$

$\ce{\implies c = \frac{n(NO2)}{V_{liquid}} = 4/5 * \frac{1}{22.4}M}$

***

#### V(NO<sub>2</sub>)/V(O<sub>2</sub>) = 4:4

$\ce{4NO2 + O2 + 2H2O -> 4HNO3}$

解析：二氧化氮与氧气按 $4:1$ 反应，剩下3份氧气占据烧瓶 $\ce{3/8}$ 的体积，所以 $\ce{V_{liquid} = 5/8aL}$

$\ce{n(HNO3) = n(NO2) = \frac{1/2a}{22.4}mol}$

$\ce{\implies c = 4/5 * \frac{1}{22.4}M}$

***

#### V(NO<sub>2</sub>)/V(O<sub>2</sub>) = 7:1

解析：4份二氧化氮完全反应，剩下三份按照 $\ce{3NO2 + H2O = 2HNO3 + NO}$ 反应

$\ce{\therefore V(NO) = 3/8 * 1/3 a = 1/8aL \implies \ce{V_{liquid} = 7/8aL}}$

$\ce{n(HNO3) = \frac{n1 + n2}{22.4} = \frac{(4/8 + 3/8*2/3)a}{22.4} = 3/4a*\frac{1}{22.4}mol}$

$\ce{\implies c = \frac{n(HNO3)}{V_{liquid}} = 6/7*\frac{1}{22.4}M}$

## 总结

综合上述解题过程可以看出，此类题目并不难，按照化方细心分析即可得出答案

最后再给出结论：$\ce{c = \frac{溶质份数}{溶解气体份数}*\frac{1}{22.4}mol}$
