# 项目说明（Readme）

欢迎来到 Anyayay's Chemistry Note！这里是项目的概览与使用指引。

- **内容范围**：覆盖原子结构、有机化学、化学实验等高中化学核心板块。
- **文件形态**：Markdown 为主，配套 PDF 版本可在 `PDF文件/` 目录中找到。
- **阅读方式**：可通过本地 VitePress 预览或直接浏览仓库中的 `.md` 文件。
- **共建**：欢迎提交 Issue/PR 纠错、补充题解或完善实验步骤。

快速开始：
1. 本地预览：`bun run docs:dev`
2. 直接阅读：从左侧目录选择章节，或访问首页按钮跳转到首章。

如果发现链接或渲染问题，请在仓库提 Issue 说明页面路径与截图。
